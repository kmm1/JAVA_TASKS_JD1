задан LinkedList представляющий собой хранилище чисел.
метод produce бесконечно добавляет в этот список случайные числа.
Однако максимальное число которое он может добавить равно 10.
метод consume бесконечно считывает числа с какой-то случайной задержкой (от 0 до 10мс).
 Сделать так чтоб метод produce добавлял числа только тогда когда не привышен лимит.
 а метод consume забирал их только тогда когда когда в списке что-то есть.
 методы должны корректно работать в многопоточной среде.
 использовать метод removeFirst для получения элемента.
 Создать и запустить 2 разных потока один из которых вызывает produce другой - concume.
 Продемонстрировать корректность работы хранилища с помощью вывода сообщений в консоль о добавлении, получении,
 и текущем размере хранилища на этапах добавления и получения.
