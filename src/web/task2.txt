1 создать get запрос отправляющий 2 URI параметра: имя и фамилию.
Сервлет принимающий запрос должен пользователю вернуть полное ФИО.
Если один из параметров не указан то сервлет должен вывести сообщение об ошибке.
2 создать post запрос с теми же условиями. Данные отправить в теле запроса двумя строками
имя на первой строке, фамилия на второй.
3 Создать мервлет который по содержимому header "User-Agent" будет определять с помощью кокого из двух браузеров (
Chrome, FireFox) вошел пользователь.
